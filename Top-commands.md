# Top SOC Commands Cheat Sheet

## Linux Basics
- `pwd` → current directory
- `ls -la` → list all files
- `cd` → change directory
- 'cat' -> to view the file contents without opening it
- 'rm' -> to remove file (-f) or directory (-r)
- 'mv' -> to move or rename the file or directory
- 'cp' -> copying
- 'whoami' -> current user
- 'id' -> uid of user, group and global
- 'chmod' -> chnages file permissions
- 'mkdir' -> creates directory
- 'touch' -> creates file
- 'uptime' -> machine running time

## Networking
- `ping 8.8.8.8`
- `ip addr`
- `ss -tuln` → listening ports
- 'ps aux' -> list of services running
- 'traceroute' -> to trace the route of one machine to another (mine)
- 'ssh' -> secure tunnel between machines
- 'ports' -> top 10 ports => http -> port 80
                             https -> port 443
                             ssh -> port 22
                             dns -> port 53
                             dhcp -> port 67/68
                             smtp -> port 25
                             ftp -> port 21
                             mysql -> port 3306
                             rdp -> port 3389
                             smb -> port 445
